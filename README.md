### 📅 Day 2: Lists, Tuples, Indexing, and Slicing
**Date**: 20/05/2026

#### 🔹 Topics Covered

| # | Topic | Details |
|---|---|---|
| 1 | **Lists** | Mutable ordered collections |
| 2 | **Tuples** | Immutable ordered collections |
| 3 | **Positive Indexing** | Accessing elements from start `(0, 1, 2, ...)` |
| 4 | **Negative Indexing** | Accessing elements from end `(-1, -2, -3, ...)` |
| 5 | **Basic Slicing** | Extracting subsequences using `[start:end]` |
| 6 | **Slicing with Step** | Using `[start:end:step]` for custom intervals |
| 7 | **Reverse Slicing** | Reversing sequences using `[::-1]` |
| 8 | **Adding Elements** | `append()`, `insert()`, `extend()` |
| 9 | **Removing Elements** | `remove()`, `pop()`, `clear()` |
| 10 | **List Methods** | Built-in methods for list manipulation |
| 11 | **Tuple Methods** | Built-in methods for tuple operations |
| 12 | **Tuple Packing/Unpacking** | Assigning and destructuring tuples |
| 13 | **Mutable vs Immutable** | Key difference between lists and tuples |

#### 💡 What I Practiced
- Creating and manipulating lists and tuples
- Accessing elements using positive and negative indexes
- Extracting data using various slicing techniques
- Adding, removing, and updating list elements
- Understanding tuple immutability
- Working with list and tuple built-in methods
- Packing and unpacking tuples

#### 🎓 Key Takeaway
Mastered sequence data structures in Python. Lists are flexible and mutable; tuples are fixed and immutable — each has its own use case in data engineering workflows.
